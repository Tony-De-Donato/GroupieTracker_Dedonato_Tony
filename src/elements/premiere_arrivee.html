PROJET GROUPIE TRACKER

Bienvenue sur mon site, ce dernier a été réalisé dans le cadre du projet "groupie tracker".
Il a pour but d'apprendre à utiliser une API.
Vous pouvez rechercher un agent en cliquant sur le bouton "COMMENCER" ci-dessous.
Vous pouvez également consulter le site officiel de valorant en cliquant sur "OFFICIAL WEBSITE".
Ou encore le site de l'api dont je me sers en cliquant sur "API UTILISEE".

Comment utiliser les requètes ?

Pour utiliser les requètes, il vous suffit de cliquer sur le bouton "COMMENCER" ci-dessous.
Ou bien sur le bouton "AGENT" dans le menu.
Vous serez redirigé vers une page où vous pourrez effectuer une recherche
Pour cela, il vous suffit de rentrer le nom de l'agent dans la barre et d'appuyer sur le bouton confirmer.

Si vous ne connaissez pas les nom des agents, ne vous en faites pas.
Il suffit de taper une seul lettre et vous aurez la liste des agents dont les noms contiennent la lettre.

Enfin, si vous n'entrez rien dans la barre de recherche, vous aurez la liste de tous les agents.
